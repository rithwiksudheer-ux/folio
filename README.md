# Folio — Clean Companion Code

These are the three working files for the ZERO2DEV Session 02 Master Kit.

## Why this folder exists
Copying code out of a Word document can break it — the document silently turns
straight quotes ( ' " ) into curly "smart" quotes ( ' ' " " ), which code rejects.
These files use clean, straight quotes and are ready to run. **Copy from here, not
from inside the books.**

## How to use
1. Put these three files in one folder (keep the names exactly as they are):
   - index.html
   - style.css
   - script.js
2. Open the folder in VS Code (or vscode.dev).
3. Right-click index.html → "Open with Live Server" — or just double-click it.
4. Edit, save, refresh, repeat.

## Before you deploy
Find-and-replace these placeholders with your real details:
- "Your Name"            → your name
- you@example.com        → your email
- https://figma.com      → your real Tide prototype link
- https://github.com     → your GitHub profile
- https://linkedin.com   → your LinkedIn profile

## Deploy
Drag this whole folder onto netlify.com's "Deploy manually" box. Done.
